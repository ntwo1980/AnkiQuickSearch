# Anki quick search

![Cloze Front](https://github.com/ntwo1980/AnkiQuickSearch/blob/main/screenshot.png?raw=true)

A simple Anki addon that adds several checkboxes and dropdown menus to the browser's search bar for quick card filtering. By default, this addon hides suspended cards.

## Features

- **Show Suspended**: When checked, suspended cards will be displayed.
- **New**: When checked, only new cards will be displayed.
- **Due**: A dropdown menu to filter cards due within a specified number of days. This is a single-selection option.
- **Studied**: A dropdown menu to filter cards studied within a specified number of days. This is a single-selection option.
- **Flag**: A multi-select dropdown menu to filter cards by flag. You can choose not to filter, show cards with any flag, or select one or more specific flags.
- **Recent Added**: When checked, it will show recently added new cards (default is the last 10 days, which can be modified in the addon configuration `recent_added_days`).
